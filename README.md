#  Diwali Sales Analysis using Python

## 📌 Project Overview

This project performs an Exploratory Data Analysis (EDA) on Diwali Sales data using Python. The objective is to understand customer purchasing behavior during the Diwali festival and extract valuable business insights that can help retailers improve sales and marketing strategies.

The project involves data cleaning, preprocessing, visualization, and business insight generation using Python libraries.


# 🎯 Business Objective

The primary objective of this project is to analyze customer purchasing patterns during the Diwali festival to help businesses:

- Identify high-value customers
- Understand customer demographics
- Discover top-selling products
- Find high-performing states
- Improve marketing campaigns
- Increase overall sales revenue



# 📂 Dataset

**Dataset Name:**
Diwali Sales Data.csv

# 📈 Key Business Insights

✔ Female customers contribute more to overall sales.

✔ Customers aged between **26–35 years** are the most active buyers.

✔ Married women have higher purchasing power during Diwali.

✔ States like Uttar Pradesh, Maharashtra, and Karnataka generate the highest revenue.

✔ IT, Healthcare, and Aviation professionals contribute significantly to sales.

✔ Food, Clothing, and Electronics are among the top-selling product categories.
